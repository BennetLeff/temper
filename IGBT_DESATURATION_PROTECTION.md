# IGBT Desaturation Protection Design

## Document Information
- **Project:** Temper Induction Cooker
- **Version:** 1.0
- **Date:** 2025-12-17
- **Status:** CRITICAL SAFETY ADDITION

---

## 1. Executive Summary

### Problem Statement
The current design lacks desaturation (DESAT) detection for the IGBTs. When an IGBT enters desaturation during a fault condition (shoot-through, load short, etc.), it conducts extremely high current (>100A) in the linear region, leading to:
- Junction temperature reaching 175°C in microseconds
- Device destruction
- Potential case rupture and arc flash
- PCB fire hazard

### Solution
Add external DESAT detection circuit that monitors collector-emitter voltage (VCE) during conduction and triggers fast shutdown if VCE exceeds threshold.

### Gate Driver Note
**IMPORTANT:** The UCC21550BDWR does **NOT** have built-in DESAT detection. UCC21551 and UCC21553 have this feature. For production, consider upgrading to UCC21551 which includes integrated DESAT detection.

## 2. Desaturation Theory

### Normal Operation vs. Desaturation

#### Normal (Saturated) Operation
```
Collector ──┬── VCE(sat) ≈ 1.7-2.5V @ rated current
            │
         [IGBT]
            │
Emitter ────┴──
```
- IGBT fully enhanced
- Low voltage drop (VCE(sat))
- Most power dissipated in load

#### Desaturation (Fault)
```
Collector ──┬── VCE >> VCE(sat), can reach 50-300V
            │
         [IGBT] ← Operating in linear region
            │     (not fully on, high loss)
Emitter ────┴──
```
- IGBT enters linear/active region
- High VCE voltage
- Massive power dissipation in IGBT: P = VCE × IC
- Junction overheats in <10µs

### Causes of Desaturation
1. **Shoot-through:** Both IGBTs conduct simultaneously
2. **Short circuit:** Load or resonant cap shorted
3. **Overcurrent:** Current exceeds SOA (Safe Operating Area)
4. **Gate drive loss:** Bootstrap failure, VGE drops below threshold
5. **Miller turn-on:** dV/dt induced false turn-on during other IGBT switching

## 3. DESAT Detection Circuit Design

### Circuit Topology

```
                                DC Bus (+340V)
                                     │
                                     ├─── R1 (1MΩ) ───┬─── D_DESAT (1200V Fast) ───┐
                                     │                │                             │
                                     │                │                             │
                                ┌────┴────┐          C_BLANK                     DESAT
                                │  IGBT   │          (100pF)                    (to comparator)
                                │   HS    │           │                             │
                                └────┬────┘           │                             │
                                     │SW          ────┴────                         │
                                     │            GND (CGND)                        │
                                                                                    │
                                                   ┌─────────────────────────────────┘
                                                   │
                                            ┌──────┴──────┐
                                            │  Comparator │  VCC = 3.3V
                                            │   TLV3201   │
                                            │  +  -  OUT  │
                                            └──┬───┬───┬──┘
                                               │   │   │
                                         V_REF─┘   │   └──── FAULT (to DIS pin & ESP32)
                                        (7.0V)    DESAT
                                                  signal

            Where V_REF is generated by resistor divider from 3.3V
```

### Component Selection

#### DESAT Diode (D_DESAT)
| Parameter | Specification |
|-----------|---------------|
| Type | Fast recovery diode or SiC Schottky |
| Voltage rating | ≥1200V (same as IGBT) |
| Current rating | ≥1A |
| Recovery time | <100ns |
| **Recommended part** | STTH1R06 (STMicroelectronics) or C3D02060E (Wolfspeed SiC) |

**Purpose:** Clamps DESAT node to VCE + V_diode when IGBT is on

#### Resistor Divider (R1)
| Parameter | Value |
|-----------|-------|
| Resistance | 1MΩ (±1%) |
| Power rating | ≥0.25W |
| Voltage rating | ≥400V per resistor (use 3× in series if needed) |
| Package | 1206 or larger |

**Purpose:** Current-limits the detection path

**Calculation:**
```
At VCE = 340V (worst case), I = 340V / 1MΩ = 340µA
P = V² / R = (340V)² / 1MΩ = 0.116W < 0.25W ✓
```

#### Blanking Capacitor (C_BLANK)
| Parameter | Value |
|-----------|-------|
| Capacitance | 100pF |
| Voltage rating | ≥100V |
| Dielectric | C0G/NP0 (low temp coefficient) |
| Package | 0603 or 0805 |

**Purpose:** Filters switching transients during IGBT turn-on

**Blanking time calculation:**
```
t_blank = R1 × C_BLANK = 1MΩ × 100pF = 100µs

Wait, that's wrong! Need to account for actual charging path.

More accurate: t_blank ≈ 3 × R_equiv × C_BLANK
Where R_equiv is parallel combination of R1 and comparator input impedance.

With comparator R_in ≈ 1MΩ:
R_equiv = 1MΩ || 1MΩ = 500kΩ
t_blank ≈ 3 × 500kΩ × 100pF = 150ns

This is appropriate to filter the IGBT turn-on transient.
```

#### Comparator (U_DESAT)
| Parameter | Specification |
|-----------|---------------|
| Type | Fast comparator with push-pull output |
| Propagation delay | <50ns |
| Supply voltage | 3.3V or 5V |
| Output | CMOS compatible |
| **Recommended part** | TLV3201AIDBVR (Texas Instruments) |
| **Alternative** | LM397 (Analog Devices, <40ns) |

**Key specifications:**
- tPD = 40ns typical (TLV3201)
- Supply: 2.7-5.5V
- Output sinks/sources 20mA (can drive LED + logic)
- Built-in hysteresis option (external resistors)

#### Reference Voltage (V_REF)
**Threshold calculation:**
```
IKW40N120H3 specifications:
- VCE(sat) typ = 2.0V @ 40A, 25°C
- VCE(sat) max = 2.5V @ 40A, 125°C

DESAT threshold selection:
- Set threshold well above VCE(sat): V_REF = 7.0V
- This provides:
  * 4.5V margin over typical VCE(sat)
  * 4.5V margin over max VCE(sat)
```

**Resistor divider for V_REF:**
```
3.3V ─── R_REF1 (10kΩ) ───┬─── V_REF (to comparator)
                          │
                      R_REF2 (4.7kΩ)
                          │
                         GND

V_REF = 3.3V × R_REF2 / (R_REF1 + R_REF2)
      = 3.3V × 4.7kΩ / (10kΩ + 4.7kΩ)
      = 1.05V

WAIT - this is too low! Need to reconsider.
```

**Revised approach using voltage divider on DESAT signal:**

Since DESAT node will be at VCE during conduction, and VCE can reach 340V during fault, we need to scale it down:

```
DESAT node ─── R_DIV1 (100kΩ) ───┬─── V_SCALED (to comparator +)
                                  │
                              R_DIV2 (2.2kΩ)
                                  │
                                 GND

At VCE = 340V (fault):
V_SCALED = 340V × 2.2kΩ / (100kΩ + 2.2kΩ) = 7.3V

But comparator supply is only 3.3V! This won't work.
```

**Correct approach - Use clamped divider:**

```
DESAT node ─── R_DIV1 (220kΩ) ───┬─── V_SCALED (to comparator +)
                                  │
                              R_DIV2 (22kΩ) ──┬─── TVS/Zener (3.0V)
                                              │
                                             GND

Scale factor = 22kΩ / (220kΩ + 22kΩ) = 0.091

At VCE(sat) = 2.5V: V_SCALED = 2.5V × 0.091 = 0.23V
At VCE(fault) = 50V: V_SCALED = 50V × 0.091 = 4.5V → clamped to 3.0V

Set V_REF = 0.7V (corresponds to VCE ≈ 7.7V threshold)
```

## 5. Detection Threshold Settings

### Threshold Voltage Calculation
```
V_REF = 3.3V × R_REF2 / (R_REF1 + R_REF2)
      = 3.3V × 1kΩ / (4.7kΩ + 1kΩ)
      = 0.58V

This corresponds to:
VCE_threshold = V_REF / scale_factor
              = 0.58V / 0.091
              = 6.4V
```

### Detection Margins
| Condition | VCE | V_SCALED | Result |
|-----------|-----|----------|--------|
| Normal saturation | 2.5V | 0.23V | No trip (< V_REF) |
| Mild desaturation | 5V | 0.45V | No trip (margin) |
| **Detection threshold** | **6.4V** | **0.58V** | **Trip point** |
| Moderate fault | 20V | 1.82V | Trip (fault detected) |
| Severe fault | 100V | 9.1V → 3.0V | Trip (clamped by TVS) |

### Blanking Time

The blanking capacitor prevents false triggering during IGBT turn-on transient:

```
During turn-on, VCE briefly exceeds threshold as IGBT enters saturation.
t_blanking ≈ 3 × R_equiv × C_BLANK
           = 3 × 500kΩ × 100pF
           = 150ns

This is sufficient to ignore turn-on transient (<100ns typically).
```

## 6. Response Time Analysis

### Fault Detection Sequence

1. **Fault occurs** (t = 0)
   - IGBT enters desaturation
   - VCE rises from 2.5V to >50V

2. **DESAT node responds** (t = 50-100ns)
   - Voltage appears at DESAT node through D_DESAT
   - Filtered by C_BLANK (150ns time constant)

3. **Comparator trips** (t = 200ns)
   - V_SCALED exceeds V_REF
   - Comparator output goes HIGH
   - Propagation delay: 40ns (TLV3201)

4. **Shutdown initiated** (t = 250ns)
   - FAULT signal asserts DIS pin on UCC21550
   - Gate drive outputs disabled
   - UCC21550 response time: 48ns

5. **IGBT turns off** (t = 500ns)
   - Gate discharges through RG_OFF
   - Collector current falls to zero
   - IGBT turn-off time: 200-400ns

**Total response time: <500ns from fault to IGBT off**

### Comparison to Alternatives

| Protection Method | Response Time | Effectiveness |
|-------------------|---------------|---------------|
| **DESAT detection** | **<500ns** | **Excellent** (catches all faults) |
| Overcurrent (CT-based) | 33ns | Good (misses desaturation if I < limit) |
| Firmware OCP | >10µs | Poor (IGBT destroyed before response) |
| Thermal shutdown | >100ms | Useless (junction already failed) |

## 7. Firmware Integration

### GPIO Configuration

```c
// ESP32-S3 GPIO assignment
#define FAULT_DESAT_HS_PIN  GPIO_NUM_4  // High-side DESAT fault input
#define FAULT_DESAT_LS_PIN  GPIO_NUM_5  // Low-side DESAT fault input
#define GATE_DIS_PIN        GPIO_NUM_6  // UCC21550 DIS pin output

// Configure as inputs with pull-down (active HIGH fault)
gpio_config_t fault_cfg = {
    .pin_bit_mask = (1ULL << FAULT_DESAT_HS_PIN) | (1ULL << FAULT_DESAT_LS_PIN),
    .mode = GPIO_MODE_INPUT,
    .pull_up_en = GPIO_PULLUP_DISABLE,
    .pull_down_en = GPIO_PULLDOWN_ENABLE,
    .intr_type = GPIO_INTR_POSEDGE  // Trigger on rising edge
};
gpio_config(&fault_cfg);

// Configure DIS output (active HIGH disables)
gpio_config_t dis_cfg = {
    .pin_bit_mask = (1ULL << GATE_DIS_PIN),
    .mode = GPIO_MODE_OUTPUT,
    .pull_up_en = GPIO_PULLUP_DISABLE,
    .pull_down_en = GPIO_PULLDOWN_DISABLE
};
gpio_config(&dis_cfg);
gpio_set_level(GATE_DIS_PIN, 0);  // Initially enabled
```

### Interrupt Handler

```c
typedef enum {
    FAULT_NONE = 0,
    FAULT_DESAT_HS = 1,
    FAULT_DESAT_LS = 2,
    FAULT_OCP = 3,
    FAULT_OVP = 4,
    FAULT_THERMAL = 5
} fault_type_t;

static fault_type_t active_fault = FAULT_NONE;
static uint32_t fault_timestamp = 0;
static uint32_t fault_count[6] = {0};

// Interrupt service routine (runs in ISR context - keep fast!)
static void IRAM_ATTR desat_fault_isr(void* arg) {
    uint32_t now = esp_timer_get_time();  // Microseconds

    // Determine which IGBT faulted
    bool hs_fault = gpio_get_level(FAULT_DESAT_HS_PIN);
    bool ls_fault = gpio_get_level(FAULT_DESAT_LS_PIN);

    if (hs_fault) {
        active_fault = FAULT_DESAT_HS;
        fault_count[FAULT_DESAT_HS]++;
    } else if (ls_fault) {
        active_fault = FAULT_DESAT_LS;
        fault_count[FAULT_DESAT_LS]++;
    }

    fault_timestamp = now;

    // Immediate shutdown (hardware DIS pin should already be asserted by comparator)
    // This is redundant backup in firmware
    gpio_set_level(GATE_DIS_PIN, 1);  // Disable gate drivers

    // Stop PWM generation
    mcpwm_stop(MCPWM_UNIT_0, MCPWM_TIMER_0);

    // Set flag for main task to handle
    xTaskNotifyFromISR(fault_handler_task, active_fault, eSetValueWithOverwrite, NULL);
}

// Install ISR
gpio_install_isr_service(0);
gpio_isr_handler_add(FAULT_DESAT_HS_PIN, desat_fault_isr, NULL);
gpio_isr_handler_add(FAULT_DESAT_LS_PIN, desat_fault_isr, NULL);
```

### Fault Handler Task

```c
void fault_handler_task(void* pvParameters) {
    uint32_t fault_code;

    while (1) {
        // Block waiting for fault notification
        if (xTaskNotifyWait(0, 0xFFFFFFFF, &fault_code, portMAX_DELAY) == pdTRUE) {

            // Log fault with timestamp
            ESP_LOGE(TAG, "DESAT FAULT detected on %s IGBT at %lu us",
                     (fault_code == FAULT_DESAT_HS) ? "HIGH-SIDE" : "LOW-SIDE",
                     fault_timestamp);

            // Diagnostic information
            ESP_LOGI(TAG, "Fault statistics:");
            ESP_LOGI(TAG, "  High-side DESAT: %lu", fault_count[FAULT_DESAT_HS]);
            ESP_LOGI(TAG, "  Low-side DESAT: %lu", fault_count[FAULT_DESAT_LS]);

            // Enter safe state
            enter_safe_state();

            // Display error on UI
            display_error("E-DESAT: IGBT FAULT");

            // Flash fault LED
            blink_fault_led(FAULT_LED_FAST);

            // Wait for user acknowledgment (button press or power cycle)
            wait_for_reset_button();

            // After reset button:
            // - Clear fault flag
            // - Re-enable gate drivers (release DIS pin)
            // - Allow restart
        }
    }
}

void enter_safe_state(void) {
    // 1. Disable all PWM outputs
    mcpwm_stop(MCPWM_UNIT_0, MCPWM_TIMER_0);

    // 2. Keep DIS asserted (gate drivers disabled)
    gpio_set_level(GATE_DIS_PIN, 1);

    // 3. Turn off cooling fan (no heat generation)
    gpio_set_level(FAN_ENABLE_PIN, 0);

    // 4. Disable relay (disconnect AC mains if used)
    gpio_set_level(RELAY_PIN, 0);

    // 5. Log fault to non-volatile storage
    nvs_set_u32(nvs_handle, "fault_count", ++total_fault_count);
    nvs_set_u32(nvs_handle, "last_fault_time", fault_timestamp);
    nvs_commit(nvs_handle);
}
```

### Fault Recovery

```c
bool attempt_fault_recovery(void) {
    ESP_LOGW(TAG, "Attempting fault recovery...");

    // Verify fault condition cleared
    vTaskDelay(pdMS_TO_TICKS(100));  // Wait 100ms

    bool hs_fault = gpio_get_level(FAULT_DESAT_HS_PIN);
    bool ls_fault = gpio_get_level(FAULT_DESAT_LS_PIN);

    if (hs_fault || ls_fault) {
        ESP_LOGE(TAG, "Fault still present, cannot recover");
        return false;
    }

    // Check IGBT health (read VCE while off - should be near VDC)
    // This would require additional ADC circuit

    // Release DIS pin
    gpio_set_level(GATE_DIS_PIN, 0);

    // Clear fault flag
    active_fault = FAULT_NONE;

    // Re-enable system with reduced power limit (50% for first 10 seconds)
    power_limit_percent = 50;
    power_limit_timer_start(10000);  // 10 second timer

    ESP_LOGI(TAG, "Fault recovery successful, operating at 50% power");
    return true;
}
```

## 8. Testing and Validation

### Bench Testing

#### Test 1: DESAT Threshold Verification
**Setup:**
1. Disconnect IGBT gate drive
2. Apply variable DC voltage to DESAT node (simulate VCE)
3. Measure comparator output

**Procedure:**
```
VCE = 0V     → FAULT = LOW (no trip)
VCE = 2.5V   → FAULT = LOW (normal VCE(sat))
VCE = 5V     → FAULT = LOW (margin)
VCE = 7V     → FAULT = HIGH (threshold exceeded) ✓
VCE = 10V    → FAULT = HIGH
```

**Pass criteria:** FAULT trips between 6-8V

#### Test 2: Blanking Time Verification
**Setup:**
1. Apply fast rising edge to DESAT node (simulate turn-on transient)
2. Measure comparator output with oscilloscope

**Procedure:**
```
Apply: 0V → 10V step (50ns rise time)
Measure: Delay from input edge to FAULT assertion
Expected: >150ns blanking time (ignores transient)
```

**Pass criteria:** Blanking time 100-200ns

#### Test 3: Response Time
**Setup:**
1. Normal operation at reduced power (500W)
2. Force shoot-through by disabling dead-time (DANGEROUS - use current limit!)
3. Measure time from fault to IGBT turn-off

**Procedure:**
```
1. Set DC bus current limit to 5A (power supply)
2. Reduce dead-time to 0ns
3. Start switching at low frequency (10kHz)
4. Trigger scope on FAULT signal
5. Measure:
   - FAULT assertion time
   - Gate voltage fall time
   - DC bus current rise time (should be limited by DESAT)
```

**Pass criteria:**
- FAULT asserts within 500ns of current rise
- Gate disabled within 1µs total

### System Integration Testing

#### Test 4: Shoot-Through Protection
**WARNING: Potentially destructive. Use sacrificial IGBTs if available.**

**Setup:**
1. Full system with IGBTs installed
2. DC bus at 50V (reduced voltage for safety)
3. Current probe on DC bus

**Procedure:**
```
1. Normal operation verification
2. Deliberately introduce very short dead-time (100ns)
3. Monitor for DESAT trip
4. Verify:
   - System shuts down before overcurrent damage
   - IGBTs survive
   - Fault logged correctly
```

#### Test 5: Load Short Circuit
**Setup:**
1. Full system, normal operation at 1kW
2. Short circuit switch across resonant capacitor (remotely actuated)

**Procedure:**
```
1. Start heating (normal operation)
2. Actuate short circuit switch
3. Verify:
   - DESAT detects fault <1ms
   - System shuts down
   - No visible damage to components
   - Fault UI displays correctly
```

### Production Testing

#### Automated Test Sequence

```
1. Power-on self-test (POST)
   - Verify DESAT circuits present (measure V_REF)
   - Check comparator outputs (should be LOW)

2. Threshold calibration check
   - Inject test signal to DESAT node
   - Verify trip point 6-8V

3. Response time verification
   - Pulse DESAT node HIGH
   - Measure FAULT assertion time <100ns

4. Full system test
   - Run at 500W for 30 seconds
   - Monitor for false trips
   - Verify FAULT remains LOW
```

**Pass/Fail Criteria:**
- All tests pass: READY FOR SHIPMENT
- Any test fails: REJECT, debug and retest

## 10. Alternative: Upgrade to UCC21551

### UCC21551 Integrated DESAT

The UCC21551 includes built-in DESAT detection, which eliminates most external components:

**Advantages:**
- Integrated detection (no external comparator)
- Programmable blanking time
- Programmable threshold
- Faster response time (< 50ns total)
- Fewer components (lower cost, higher reliability)

**Disadvantages:**
- More expensive than UCC21550 (+$0.50/unit)
- Requires design re-validation
- Different pin configuration

**Recommendation for production:** Consider UCC21551 for next revision to simplify BOM and improve reliability.
