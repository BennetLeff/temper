# TLV3201 - High-Speed Rail-to-Rail Comparator

**Description:** The TLV3201 is a single-channel, high-speed comparator with rail-to-rail input and output capabilities. It features a fast propagation delay of 40ns and is designed for low-voltage applications.

**Key Specifications:**
- Propagation Delay: 40ns
- Supply Voltage: 2.7V to 5.5V
- Quiescent Current: 40µA
- Package: SOT-23-5
- Input Type: Rail-to-Rail
- Output Type: Push-Pull

**Pinout (SOT-23-5):**
1. OUT - Output
2. GND - Ground
3. IN+ - Non-inverting input
4. IN- - Inverting input
5. VCC - Supply voltage

**Application in Temper:**
Used in the safety interlock system for:
- Overcurrent Protection (OCP)
- Overvoltage Protection (OVP)
- Thermal Shutdown

- Manufacturer: Texas Instruments
- MPN: TLV3201AIDBVR
