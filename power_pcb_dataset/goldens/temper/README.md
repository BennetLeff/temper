# Golden DSN Fixtures for temper board

These golden DSN files are generated by running the full pipeline and
exporting DSN at each stage boundary.

## Generation

```bash
# Generate all golden files
temper dsn generate --input pcb/temper.kicad_pcb --config pcb/temper_config.yaml

# Generate a single boundary
temper dsn generate --boundary placement --input pcb/temper.kicad_pcb --config pcb/temper_config.yaml
```

## Verification

```bash
# Check all boundaries against goldens
temper dsn check --boundary placement --input pcb/temper.kicad_pcb --config pcb/temper_config.yaml
```

Golden files must be regenerated when the pipeline output schema changes.
